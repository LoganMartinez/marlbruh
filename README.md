# marlbruh

frontend setup

- install nodejs
- go to frontend-marlbruh directory
- npm install
- npm run dev

backend setup

- python3 -m venv venv
- source ./venv/bin/activate
- python3 -m pip install -r backend-marlbruh/requirements.txt
- go to backend-marlbruh directory
- python3 manage.py migrate
- python3 manage.py runserver
